Getting Started

An overview of the Collab UI Toolkit, how to download and use, examples, and more.

Install with Yarn or NPM

Install and manage the Collab UI Toolkit's Sass, CSS, JavaScript, and fonts using NPM.

In your project's .yarnrc or .npmrc file, add the following entry:

registry=http://engci-maven-master.cisco.com/artifactory/api/npm/collab-ui-npm-group

Install the packages with yarn add. By default, yarn installs packages to node_modules/.

yarn add collab-ui

(for Angular 2+)

yarn add @collab-ui/angular

(for ReactJS)

yarn add collab-ui-react

(for Angular 1.X)

yarn add collab-ui-ng

(for Angular Formly)

yarn add collab-ui-formly
